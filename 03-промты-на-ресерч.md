# Промты на ресерч

Эти два промта я гонял, чтобы собрать шпаргалку для нашего менеджера
по сегменту «поставщики расходников и комплектующих для промоборудования».
Публикую как есть, чтобы коллеги могли собрать такую же для своего
сегмента.

## Промт 1. Собрать сегмент

Скинь мне короткую справку по b2b-сегменту «поставщики расходников и
комплектующих для промышленного оборудования» в РФ. Цель — дать
менеджеру контекст, чтобы он не выглядел дураком на первой встрече
с клиентом из этой ниши.

Что нужно:
- типичный профиль клиента: размер, ассортимент, средний чек, штат
  менеджеров
- 5–7 ключевых игроков и конкурентов с пояснением, зачем менеджеру о
  них знать
- где живут лиды: b2b-маркетплейсы (Авито, Пульс цен, Закупай,
  Супл.Биз, DirectIndustry), свои сайты, мессенджеры
- типичные боли первой линии продаж, формулировками клиента
- какие CRM используют (amoCRM, Битрикс24, отраслевые)
- словарик терминологии клиента: жаргон, сокращения
- цикл сделки по категориям товаров

Правила:
- только то, что встречается в открытых источниках
- если не уверен, не пиши
- без энциклопедий, одна плотная страница
- цифры собрать в ASCII-блок, для каналов и игроков таблицы, боли
  списком

## Промт 2. Вытащить подходящие кейсы Botamin

Источники, в которых копаем:

- https://botamin.ru
- https://companies.rbc.ru/id/1247700620364-botamin
- https://vc.ru/id418032

Цель — вытащить из этих трёх источников реальные кейсы, которые
зайдут менеджеру на встрече с клиентом из b2b-расходников.

По каждому кейсу верни:
- клиент: отрасль, размер бизнеса, канал лидов
- боль клиента до внедрения, формулировкой клиента где есть
- что сделали, без жаргона LLM, простыми словами
- цифры результата (CR, SLA, средний чек, объём)
- цитата клиента, если есть в источнике
- URL источника

Критерии отбора:
- похожий канал лидгенерации на наш сегмент (Авито, мессенджер, b2b)
- похожий тип клиента (b2b, малый или средний бизнес, не edtech/hr)
- наличие конкретных цифр
- возможность перенести логику внедрения

Главное правило: не выдумывай. Если данных нет, так и пиши. Лучше
меньше кейсов с реальными цифрами, чем больше с придуманными.

Формат ответа:
- таблица-сводка всех найденных кейсов с одной строкой про каждый
- подробные карточки по каждому релевантному
- в конце каждой карточки ссылка на источник
